# Bank Management System

A role-based banking management system built using Next.js and Supabase.

## Features

* Customer Management
* Account Management
* Transaction Processing
* Loan Management
* Role-Based Access Control
* Audit Logging

## User Roles

### Customer

* View Account
* Deposit Money
* Withdraw Money
* Transfer Money
* Apply for Loans

### Admin

* Manage Customers
* Manage Accounts
* Monitor Transactions

### Manager

* Review Loans
* Approve Loans
* Reject Loans

## Tech Stack

Frontend:

* Next.js
* TypeScript
* Tailwind CSS

Backend:

* Next.js API Routes

Database:

* PostgreSQL (Supabase)

Authentication:

* Supabase Auth
